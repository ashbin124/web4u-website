# Interview Readiness Checklist

## Functional

- [ ] Verify all nav links on every page
- [ ] Test WhatsApp and phone links on mobile
- [ ] Submit contact form and verify email delivery
- [ ] Confirm `404.html` works on deployed host

## UX and Accessibility

- [ ] Keyboard tab through all interactive elements
- [ ] Check form labels and placeholders are clear
- [ ] Validate color contrast in Lighthouse
- [ ] Confirm focus states are visible

## Performance

- [ ] Run Lighthouse on home and contact pages
- [ ] Replace heavy stock images with compressed WebP versions
- [ ] Keep CLS low by setting fixed image dimensions

## SEO

- [ ] Replace `https://your-domain.com` placeholders
- [ ] Submit `sitemap.xml` in Google Search Console
- [ ] Validate OG preview using social preview tools

## Presentation

- [ ] Prepare 2-minute project walkthrough
- [ ] Explain one design decision and one engineering decision
- [ ] Show before/after improvements made during iteration
